Bootstrap 5 Sass Customization

To customize Bootstrap 5 with Sass, set up a project using npm, modify the _variables.scss file to change primary colors and border radius, then recompile Bootstrap. 

// Example _variables.scss customization
$primary: #ff5733;
$border-radius: 1rem;

After customization, run npm run build-css or equivalent to compile the Sass files.
